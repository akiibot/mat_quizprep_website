Trig Integration & Reduction Formulas

¬ß7.3 ¬∑ Integration of Trigonometric Functions
What is a Reduction Formula?

A reduction formula expresses \(\int \sin^n x\,dx\) (or \(\cos^n x\)) in terms of a simpler integral with a lower power. You apply it repeatedly until you reach a base case.

\[\int \sin^n x\,dx = -\frac{\sin^{n-1}x\cos x}{n} + \frac{n-1}{n}\int \sin^{n-2}x\,dx\]
\[\int \cos^n x\,dx = \frac{\cos^{n-1}x\sin x}{n} + \frac{n-1}{n}\int \cos^{n-2}x\,dx\]
üí° Both formulas have the same structure. Each application reduces the power by 2, so you keep going until you reach \(n=1\) or \(n=0\).
Base Cases to Memorize
IntegralResult
\(\int \sin x\,dx\)\(-\cos x + C\)
\(\int \cos x\,dx\)\(\sin x + C\)
\(\int \sin^0 x\,dx = \int 1\,dx\)\(x + C\)
\(\int \cos^0 x\,dx\)\(x + C\)
Worked Example ‚Äî \(\int \sin^4 x\,dx\)
  1. Apply the reduction formula with \(n=4\):
    \[\int \sin^4 x\,dx = -\frac{\sin^3 x\cos x}{4} + \frac{3}{4}\int \sin^2 x\,dx\]
  2. Apply again with \(n=2\):
    \[\int \sin^2 x\,dx = -\frac{\sin x\cos x}{2} + \frac{1}{2}\int 1\,dx = -\frac{\sin x\cos x}{2} + \frac{x}{2}\]
  3. Substitute back:
    \[\int \sin^4 x\,dx = -\frac{\sin^3 x\cos x}{4} + \frac{3}{4}\!\left(-\frac{\sin x\cos x}{2}+\frac{x}{2}\right) + C\]
Key Trig Identities You'll Need
IdentityUse
\(\sin^2 x = \frac{1-\cos 2x}{2}\)Even powers of sin
\(\cos^2 x = \frac{1+\cos 2x}{2}\)Even powers of cos
\(\sin^2 x + \cos^2 x = 1\)Odd powers ‚Äî save one factor
\(\tan^2 x = \sec^2 x - 1\)Powers of tan
üí° For odd powers (like \(\sin^5 x\)), factor out one \(\sin x\), convert the rest using \(\sin^2 x = 1-\cos^2 x\), then use substitution \(u=\cos x\).
Strategy Summary
  • Even power of sin or cos alone: Use reduction formula or half-angle identities
  • Odd power: Save one trig factor, convert rest, u-substitution
  • Mixed \(\sin^m x \cos^n x\): If one is odd, use substitution
  • tan/sec combos: Use \(\tan^2 x = \sec^2 x - 1\) and save sec¬≤x for du

Riemann Sums

¬ß5.4 ¬∑ Approximating Definite Integrals
Core Idea

A Riemann sum approximates \(\int_a^b f(x)\,dx\) by dividing \([a,b]\) into \(n\) subintervals of equal width and summing rectangle areas.

\[\Delta x = \frac{b-a}{n}, \qquad \int_a^b f(x)\,dx \approx \sum_{i=1}^{n} f(x_i^*)\,\Delta x\]

where \(x_i^*\) is a sample point in the \(i\)-th subinterval.

Three Types of Sample Points
TypeSample point \(x_i^*\)Formula
Left Riemann SumLeft endpoint\(x_i^* = a + (i-1)\Delta x\)
Right Riemann SumRight endpoint\(x_i^* = a + i\,\Delta x\)
Midpoint RuleMidpoint\(x_i^* = a + \left(i-\tfrac{1}{2}\right)\Delta x\)
üí° The midpoint rule is generally the most accurate of the three.
Worked Example

Approximate \(\int_1^3 x^2\,dx\) using a Right Riemann Sum with \(n=4\).

  1. \(\Delta x = \frac{3-1}{4} = 0.5\)
  2. Right endpoints: \(x_1=1.5,\ x_2=2.0,\ x_3=2.5,\ x_4=3.0\)
  3. Evaluate: \(f(1.5)=2.25,\ f(2)=4,\ f(2.5)=6.25,\ f(3)=9\)
  4. Sum: \((2.25+4+6.25+9)\times 0.5 = 21.5\times 0.5 = \mathbf{10.75}\)
  5. Exact value: \(\left[\frac{x^3}{3}\right]_1^3 = 9-\frac{1}{3} \approx 8.67\). The right sum overestimates here since \(f\) is increasing.
Over/Under-Estimation Rules
Function is‚Ä¶Left SumRight Sum
Increasing on \([a,b]\)UnderestimateOverestimate
Decreasing on \([a,b]\)OverestimateUnderestimate
‚ö†Ô∏è If the function changes monotonicity, you cannot make a simple over/under claim for the whole interval.
The Exact Integral as a Limit
\[\int_a^b f(x)\,dx = \lim_{n\to\infty}\sum_{i=1}^{n} f(x_i^*)\,\Delta x\]

As \(n\to\infty\), all three sums converge to the same value (the exact integral) for any continuous function.

Improper Integrals

¬ß7.8 ¬∑ Infinite Limits or Discontinuous Integrands
What Makes an Integral "Improper"?
  • Type 1: One or both limits of integration are \(\pm\infty\)
  • Type 2: The integrand has a vertical asymptote (discontinuity) somewhere on \([a,b]\)
Type 1 ‚Äî Infinite Limits
\[\int_a^{\infty} f(x)\,dx = \lim_{t\to\infty}\int_a^t f(x)\,dx\]
\[\int_{-\infty}^b f(x)\,dx = \lim_{t\to-\infty}\int_t^b f(x)\,dx\]
\[\int_{-\infty}^{\infty} f(x)\,dx = \int_{-\infty}^c f\,dx + \int_c^{\infty} f\,dx\]
üí° Split \(\int_{-\infty}^{\infty}\) at any convenient \(c\) (usually 0). The integral converges only if both parts converge.
Type 2 ‚Äî Discontinuous Integrand

If \(f\) has a discontinuity at \(x=a\):

\[\int_a^b f(x)\,dx = \lim_{t\to a^+}\int_t^b f(x)\,dx\]

If the discontinuity is at an interior point \(c \in (a,b)\):

\[\int_a^b f(x)\,dx = \int_a^c f\,dx + \int_c^b f\,dx\]
‚ö†Ô∏è Always check for hidden discontinuities! E.g. \(\int_{-1}^{1}\frac{1}{x}\,dx\) ‚Äî don't just plug in endpoints; there's a discontinuity at \(x=0\).
The p-Integral (Must Know!)
\[\int_1^{\infty}\frac{1}{x^p}\,dx = \begin{cases} \dfrac{1}{p-1} & \text{if } p > 1\ (\text{converges}) \\ \infty & \text{if } p \le 1\ (\text{diverges}) \end{cases}\]
\[\int_0^{1}\frac{1}{x^p}\,dx = \begin{cases} \dfrac{1}{1-p} & \text{if } p < 1\ (\text{converges}) \\ \infty & \text{if } p \ge 1\ (\text{diverges}) \end{cases}\]
üí° Notice the conditions are flipped for the two cases. The boundary behavior at \(\infty\) vs. at \(0\) is opposite.
Comparison Test

If \(0 \le f(x) \le g(x)\) for all \(x \ge a\):

  • If \(\int_a^\infty g\,dx\) converges ‚Üí \(\int_a^\infty f\,dx\) converges
  • If \(\int_a^\infty f\,dx\) diverges ‚Üí \(\int_a^\infty g\,dx\) diverges

Compare unknown integrals to p-integrals or exponentials.

Worked Example ‚Äî \(\int_1^{\infty} e^{-x}\,dx\)
  1. Replace \(\infty\) with \(t\): \(\displaystyle\lim_{t\to\infty}\int_1^t e^{-x}\,dx\)
  2. Integrate: \(\displaystyle\lim_{t\to\infty}\left[-e^{-x}\right]_1^t = \lim_{t\to\infty}(-e^{-t}+e^{-1})\)
  3. Take the limit: \(= 0 + e^{-1} = \dfrac{1}{e}\)
  4. Converges to \(1/e\).

Rational Functions (Partial Fractions)

¬ß7.5 ¬∑ Integration of Rational Functions
When to Use Partial Fractions

Use this method to integrate a rational function \(\dfrac{P(x)}{Q(x)}\) where \(\deg P < \deg Q\).

‚ö†Ô∏è If \(\deg P \ge \deg Q\), do polynomial long division first, then apply partial fractions to the remainder.
Decomposition Cases
Factor in \(Q(x)\)Partial Fraction Term(s)
Linear: \((ax+b)\)\(\dfrac{A}{ax+b}\)
Repeated linear: \((ax+b)^k\)\(\dfrac{A_1}{ax+b}+\cdots+\dfrac{A_k}{(ax+b)^k}\)
Irreducible quadratic: \(ax^2+bx+c\)\(\dfrac{Ax+B}{ax^2+bx+c}\)
Repeated irreducible quad: \((ax^2+bx+c)^k\)One term per power, numerator \(Ax+B\)
Step-by-Step Method
  1. Check degree ‚Äî if improper, do long division first
  2. Factor the denominator \(Q(x)\) completely
  3. Write the partial fraction decomposition (see table above)
  4. Multiply both sides by \(Q(x)\) to clear denominators
  5. Solve for constants: plug in convenient values of \(x\) (roots of \(Q\)) and/or match coefficients
  6. Integrate each term ‚Äî typically gives \(\ln|x+a|\) or \(\arctan\)
Worked Example ‚Äî \(\int\dfrac{1}{x^2-1}\,dx\)
  1. Factor: \(x^2-1 = (x-1)(x+1)\)
  2. Write: \(\dfrac{1}{(x-1)(x+1)} = \dfrac{A}{x-1}+\dfrac{B}{x+1}\)
  3. Multiply by \((x-1)(x+1)\): \(1 = A(x+1)+B(x-1)\)
  4. Plug in \(x=1\): \(1=2A \Rightarrow A=\frac{1}{2}\). Plug in \(x=-1\): \(1=-2B \Rightarrow B=-\frac{1}{2}\)
  5. Integrate: \(\dfrac{1}{2}\int\dfrac{1}{x-1}dx - \dfrac{1}{2}\int\dfrac{1}{x+1}dx = \dfrac{1}{2}\ln|x-1|-\dfrac{1}{2}\ln|x+1|+C\)
  6. Simplify: \(\dfrac{1}{2}\ln\!\left|\dfrac{x-1}{x+1}\right|+C\)
Key Integration Results
FormIntegral
\(\int\dfrac{1}{x+a}\,dx\)\(\ln|x+a|+C\)
\(\int\dfrac{1}{(x+a)^n}\,dx\)\(\dfrac{(x+a)^{1-n}}{1-n}+C\), \(n\ne1\)
\(\int\dfrac{1}{x^2+a^2}\,dx\)\(\dfrac{1}{a}\arctan\dfrac{x}{a}+C\)
\(\int\dfrac{x}{x^2+a^2}\,dx\)\(\dfrac{1}{2}\ln(x^2+a^2)+C\)
